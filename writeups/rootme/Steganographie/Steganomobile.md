# [Steganomobile](https://www.root-me.org/fr/Challenges/Steganographie/Steganomobile)

> Après avoir extrait les données d’un téléphone mobile retrouvé sur les lieux d’un crime, les enquêteurs ont récupéré cette suite de chiffres... Peut-être un numéro de téléphone ?

```
222-33-555-555-7-44-666-66-33
```

To resolve the challenge, we have to think of the old mobile keyboard style.

![keyboard](./images/keyboard.jpg)

> Image from [android.stackexchange.com](https://android.stackexchange.com/questions/45598/is-there-any-old-style-keyboard-numpad-style-available-for-android)

Let's interpret the numbers as inputs.

- 222 : C
- 33 : E
- 555 : L
- 555 : L
- 7 : P
- 44 : H
- 666 : O
- 66 : N
- 33 : E

The word we're looking for is `cellphone`.
